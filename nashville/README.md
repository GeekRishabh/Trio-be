## Description

Web API(Backend Facing Frontend) for handling REST and WebSocket the codename is
Nashville.

## Installation

Copy the .env.example as .env and update the environment values 

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
